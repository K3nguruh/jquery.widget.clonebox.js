Clonebox - Standard
Beispiel HTML-Code (stark vereinfacht)
<div data-plugin="clonebox">
  <div data-clone="row">
    <input name="input[]">
    <button data-clone-btn="add">+</button>
    <button data-clone-btn="del">-</button>
  </div>
  <div>
    <button data-clone-btn="reset">Reset</button>
  </div>
</div>

<div data-plugin="clonebox" data-limit="5">

Beispiel HTML-Code 
<button id="myID" type="button">Reset</button>
<script>
  $("#myID").on("click", function () {
    $("body").find('[data-plugin="clonebox"]').clonebox("reset");
  });
</script>
